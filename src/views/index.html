Spoke Backend Coding Challenge

The following routes are set as follows

Get routes

  • Get all tasks https://spoke-david.herokuapp.com/tasks
  • Filter tasks based on status, here you will need to change in progress to completed and vice versa https://spoke-david.herokuapp.com/tasks?status=in progress
  • Get single task with Idhttps://spoke-david.herokuapp.com/tasks/1

Put Routes

  • For the put request you will need to use Postman, you can update the task description or it's status. Example as follows

    • const body = { "id": 1, "task": "This task is done now", "status": "completed" };

    await fetch('https://spoke-david.herokuapp.com/tasks/update',  {
     method: 'PUT',
     headers: {
      'Content-Type': 'application/json'
     },
     body: JSON.stringify(body)
    });

Post Routes

  • For the post request you will need to use Postman, you can create a new task and set it's status, by default it will be in progress. Example as follows

    • const body = { "task": "I'd like to create a new task" };

    await fetch('https://spoke-david.herokuapp.com/tasks',  {
     method: 'POST',
     headers: {
      'Content-Type': 'application/json'
     },
     body: JSON.stringify(body)
    });

Delete Routes

  • For the delete request you will need to use Postman, you can delete a task with it's corresponding Id. Example as follows

    • const body = { "id": 3 };

    await fetch(`https://spoke-david.herokuapp.com/tasks/${body.id}`,  {
     method: 'DELETE',
    });
